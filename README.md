# Gacha-Widget

## 개요 (Overview & Concept)

**Gacha Machine**은 단순한 기능을 넘어, 데스크탑 환경에 재미와 무작위성을 더하는 독특한 유틸리티 애플리케이션입니다. 메인 '가챠 머신' 아이콘을 **클릭할 때마다** 다양한 기능과 개성을 가진 위젯들이 무작위로 스폰(Spawn)됩니다. 이 위젯들은 사용자의 메모, 시간 확인, 심지어 가상의 생명체 관리까지 담당하며, 책상 위를 작은 디지털 놀이터로 만들어 줍니다.

## ⚙️ 주요 위젯 및 상호작용 (Key Widgets & Interactions)

| 위젯 종류 | 기능 및 상호작용 | 특이 사항 | 
 | ----- | ----- | ----- | 
| **Gacha Machine (메인)** | 클릭하여 위젯 스폰, 우클릭 메뉴(초기화/종료) 제공. | 항상 최상위에 유지됩니다. | 
| **Post-It (포스트잇)** | 메모를 기록할 수 있는 투명한 텍스트 편집기. | 배경의 이미지/색상과 분리되어 투명하게 텍스트만 입력 가능합니다. | 
| **Clock (시계)** | 현재 시스템 시간을 표시하는 작은 디지털 시계. | 드래그하여 자유롭게 배치 가능합니다. | 
| **Cookie (포춘 쿠키)** | **클릭 시** 쿠키가 열리며, 내부에 숨겨진 무작위 **명언이나 문구**가 나타납니다. | 한 번 열린 쿠키는 닫히지 않습니다. | 
| **Asparagus (아스파라거스)** | 거북이(Turtle) 위젯의 **주요 먹이** 아이템. | 거북이가 배고플 때 아스파라거스에 닿으면 섭취되고 거북이의 배고픔이 해소됩니다. | 
| **Turtle (거북이)** | **배고픔** 상태를 가지며, 일정 시간 아스파라거스를 먹지 않으면 사망할 수 있는 펫 위젯입니다. | **Ball** 위젯과 충돌 시 즉시 사망 처리됩니다. | 
| **Ball (공)** | 터틀 위젯에게 **위험한 아이템**입니다. | 터틀과 충돌 시 터틀을 사망 상태로 만듭니다. | 
| **Cloud (구름)** | 단순히 떠다니며 데스크탑 분위기를 조성하는 위젯입니다. | 기능성 없이 미적인 역할만 수행합니다. | 
| **Black Hole (블랙홀)** | 다른 위젯들을 흡수하여 화면을 정리하는 **'휴지통'** 역할의 위젯. | 위젯을 드래그하여 블랙홀과 겹치게 하면 해당 위젯은 소멸됩니다. | 

## 🕹️ 사용 방법 (How to Use)

1. **위젯 스폰**: 메인 **Gacha Machine** 아이콘을 마우스 **왼쪽 버튼으로 클릭**합니다. 화면에 무작위 위젯이 나타납니다.

2. **위젯 이동**: 스폰된 모든 위젯(메인 머신 포함)은 드래그하여 원하는 위치로 자유롭게 이동할 수 있습니다.

3. **위젯 상호작용**:

   * **Post-It**을 클릭하여 메모를 시작합니다.

   * **Cookie**를 클릭하여 숨겨진 메시지를 확인합니다.

   * **Turtle**에게 **Asparagus**를 드래그하여 먹이를 줍니다.

4. **위젯 삭제 및 정리**:

   * **Black Hole** 위젯을 드래그하여 다른 위젯(터틀 포함, 블랙홀끼리도 가능)과 **겹치게 하면** 해당 위젯은 블랙홀에 흡수되어 사라집니다.

   * 두 개의 **Black Hole** 위젯이 충돌하면 하나가 사라져 화면에 블랙홀이 하나만 남도록 합니다.

5. **전체 초기화**: 메인 **Gacha Machine** 아이콘을 마우스 **오른쪽 버튼으로 클릭**하여 '초기화'를 선택하면 화면의 **모든 위젯**이 즉시 사라집니다.

## 🛠️ 개발 환경 및 PyInstaller 빌드

이 애플리케이션은 **Python** 언어와 **PyQt6** 프레임워크를 기반으로 개발되어, 플랫폼 독립적인 GUI 환경을 제공합니다. Windows 환경에서 배포 및 실행의 용이성을 위해 **PyInstaller**를 사용하여 단일 실행 파일(`.exe`)로 빌드됩니다.

모든 이미지(`png`, `ico`) 파일은 프로젝트의 루트에 위치한 **`assets`** 폴더에 모여 있으며, 이는 PyInstaller가 리소스를 효율적으로 관리하고 번들에 포함시키기 위한 최적의 구조입니다.

### 실행 파일 생성 명령어 (Windows 기준)
